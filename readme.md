- sway
- zsh

- foot
- neovim
- firefox (userchrome + about:config settings)

- bemenu
- grim, slurp, wl-copy

- add cursors in `/usr/share/icons/default`, `~/.config/gtk-3.0/settings.ini`, and set with `gsettings`
